Setup LuaStudio

You can run luastudio in application mode using command line arguments

On Windows

LuaStudio.exe -a Samples.luastudio /HelloWorld.lua "Custom Window Title"

On Mac

open -na LuaStudio.app --args -a Samples.luastudio /HelloWorld.lua "Custom Window Title"
