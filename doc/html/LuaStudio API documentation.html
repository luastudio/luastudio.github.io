    Introduction

    API Documentation for LuaStudio version 2.9.0


    LuaStudio - cross-platform software development tool, primarily focusing on providing access for developers to computer graphics from Lua programming language.


    LuaStudio* 2.9.0 based on Lua programming language** version 5.2. You can read more about Lua here: http://www.lua.org. 

    Lua 5.2 Reference Manual available from this URL: https://www.lua.org/manual/5.2/.

    LuaStudio JIT based on LuaJIT*** version 2.1.0-beta3 (latest from https://github.com/LuaJIT/LuaJIT). You can read more about LuaJIT here: http://luajit.org/. 


    This API documentation guides you to LuaStudio library “Lib”. All available objects, methods and properties for this library collected and documented here. 


    LuaStudio API - subject of any changes at any time.
