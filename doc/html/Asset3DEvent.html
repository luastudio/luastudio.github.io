Asset3DEvent

Dispatched whenever a ressource (asset) is parsed and created completly.


Inherited from Event.


Can be created using method new:

Lib.Away3D.Events.Asset3DEvent.new(type, asset, prevName)

Creates Asset3DEvent object 

type - String

asset - IAsset (default = nil): AnimationNodeBase, AnimationSetBase, AnimatorBase, BitmapDataAsset, EffectMethodBase, Geometry, LightPickerBase, MaterialBase, Mesh, ObjectContainer3D, ShadowMapMethodBase, Skeleton, SkeletonPose, TextureProxyBase

prevName - String (default = nil)


Created Asset3DEvent object has properties:

asset

[read-only] IAsset: AnimationNodeBase, AnimationSetBase, AnimatorBase, BitmapDataAsset, EffectMethodBase, Geometry, LightPickerBase, MaterialBase, Mesh, ObjectContainer3D, ShadowMapMethodBase, Skeleton, SkeletonPose, TextureProxyBase

assetPrevName

[read-only] String
